# Grade 9 English — Working Revision Summary

> Note: the English mind map image is very tall and less text-readable than the others, so this file organises the most likely Grade 9 English revision areas into a clear working structure in English.

## 1. Reading skills
- Identify explicit and implicit meaning.
- Track how a writer presents ideas, attitudes and feelings.
- Select short quotations accurately.
- Zoom in on keywords and explain connotations.
- Analyse language techniques such as metaphor, simile, imagery, repetition and contrast.
- Analyse structural choices such as openings, endings, shifts, flashbacks and sentence length.
- Compare viewpoints, methods and tone across texts.

## 2. Writing skills
### Narrative / descriptive writing
- Plan a clear beginning, middle and ending.
- Use varied sentence lengths.
- Build atmosphere with sensory description.
- Control viewpoint and tense consistently.
- Use punctuation accurately.

### Transactional / non-fiction writing
- Write to argue, explain, persuade, advise or review.
- Match tone to audience and purpose.
- Use topic sentences and clear paragraphing.
- Include evidence, examples and developed explanation.

## 3. Literary analysis
- Use PEEL / PEE paragraphs.
- Make a clear point.
- Support it with quotation or precise textual reference.
- Explain how the language or structure creates meaning.
- Link back to the question.

## 4. Grammar and accuracy
- Secure sentence punctuation.
- Use commas, colons, semicolons and apostrophes correctly.
- Avoid run-on sentences and fragments.
- Check subject-verb agreement.
- Improve spelling of common academic vocabulary.

## 5. Vocabulary and style
- Learn ambitious but precise vocabulary.
- Avoid vague words like ‘nice’, ‘good’, ‘bad’.
- Use verbs and adjectives with clear effect.
- Edit for clarity and fluency.

## 6. Exam habits
- Read the question carefully.
- Underline command words.
- Plan before writing.
- Leave time to proofread.
- In analysis questions, explain effect rather than only naming the technique.

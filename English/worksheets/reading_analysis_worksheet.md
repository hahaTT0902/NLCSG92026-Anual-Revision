# English Worksheet — Reading Analysis

## Section A — Skills
1. Define metaphor.
2. Define simile.
3. What is tone?
4. What is structure?

## Section B — Practice prompts
Read a short extract and answer:
1. What is the writer’s attitude?
2. Find one quotation that creates tension.
3. Explain the effect of one word choice.
4. Explain one structural feature.

## Section C — Writing task
Write one PEEL paragraph answering:
"How does the writer create a sense of fear?"
